Melhores Projetos


1. 4Defenses

É um jogo de tower defense e seu objetivo é proteger a base dos inimigos e manter-se vivo.
Este projeto foi o projeto que mais investi em horas e energia para construí-lo. Elaborei as ideias e mecânicas que queria para colocar em prática tudo o que aprendi até então.
Peguei as sprites e os assets de áudio para manter meu foco na programação e implementação de cada mecânica.
Curiosidade: Levei mais de 150 horas para concluir esse projeto sozinho.

Conhecimento adquirido:
Aplicações matemáticas.
Não gaste várias horas desenvolvendo o mesmo recurso. Pare, desenvolva outro e depois volte para tentar novamente.
Escalabilidade(boas práticas para manter o código do projeto estruturado).
Balanceamento do Jogo.

Desenvolvido: Unity2D

Funcionalidades em Destaque

O lança-chamas inicia um estado de queima por segundo nos inimigos.

Para fazer o efeito de lança-chamas utilizando duas sprites de fogo, tive que implementar cálculos trigonométricos para que cada sprite gerado fosse movido para a direção que o mouse está apontando.

Todo o jogo foi feito em torno de um sistema de ondas que gera alguns inimigos a cada rodada.
