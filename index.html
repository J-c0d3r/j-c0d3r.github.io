1. 4Defenses

It's a tower defense game and your objective is to protect the base from the enemies and keep yourself alive.
This project was the project that I spent more hours and energy to built it. I elaborated the ideas and mechanics that I want in order to put in practice all that I had learned until then.
I got the sprites and audio assets in order to keep my focus on programming and implementation of each features.
Curiosity: I took over 150 hours to complete this project alone.

Knowledge Adquired:
Mathematics Applications.
Don't spend several hours in developing the same feature. Stop, develop another, and then come back to try again.
Scalability(good practices to keep project's code structured).
Game Balance.

Built in: Unity2D

Highlights Features

The flamethrower starts a burning state per second in enemies.

To make the effect of flamethrower using two sprites of fire, I had to implement trigonometric calculations so that each generated bullet moved to the direction that the mouse is pointing.

The entire game was made around a wave system which it generates some enemies in each round.
